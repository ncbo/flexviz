Classpublic class NCBOSearchResultConcept
InheritanceNCBOSearchResultConcept NCBOItem

Simplified version of NCBOConcept which only contains the concept ID, preferred name, and the contents that matched the query. It also references the NCBOSearchResultOntology that it belongs to. And also contains the search text.
