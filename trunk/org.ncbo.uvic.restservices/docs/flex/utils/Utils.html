Packageflex.utils
Classpublic final class Utils

Utility class which contains useful functions for: - getting application, browser URL, and loader URL parameters - changing browser title - converting strings into ints, floats, booleans - preventing HTTPService objects from caching results - getting a simple classname (no package)

getParam()method 
public static function getParam(paramName:String, defaultValue:String = null, parameters:Object = null, checkBrowserURL:Boolean = false, checkLoaderURL:Boolean = false):String

Returns an application parameter. If the parameter doesn't exist then the defaultValue is returned. If the parameter doesn't exist in the application, then it can optionally try to get the parameter from the browser url.

Parameters
paramName:String — the parameter name
 
defaultValue:String (default = null) — the default value if the parameter doesn't exist
 
parameters:Object (default = null) — optionally you can pass in the application parameters, otherwise it defaults to Application.application.parameters
 
checkBrowserURL:Boolean (default = false) — if the application doesn't have the parameter, and this value is true then the browser url is parsed to see if the parameter is defined there.
 
checkLoaderURL:Boolean (default = false) — if the application doesn't have the parameter, and this value is true then the loader url (e.g. "test.swf?param=name") is parsed for parameters.

Returns
String
