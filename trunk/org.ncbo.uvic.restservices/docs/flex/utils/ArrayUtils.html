combine()method 
public static function combine(a1:Array, a2:Array, allowDuplicates:Boolean = true, newArray:Boolean = false, allowNullValues:Boolean = false):Array

Adds all the elements from the second array into the first. If you specify that no duplicates are allowed, then the resulting array will not have duplicates. If you specify true for "newArray" then a new array is returned containing the elements from both Arrays.

Parameters
a1:Array — the first array
 
a2:Array — the second array
 
allowDuplicates:Boolean (default = true) — if false then duplicate entries won't be added. Note that duplicates aren't checked for in the existing a1 array.
 
newArray:Boolean (default = false) — if true then a new array is created and returned, and the input arrays aren't changed
 
allowNullValues:Boolean (default = false) — if true then null values are allowed in the output array

Returns
Array — the same a1 array that now contains a2's elements as well, OR a new array if newArray is true

Adds all the elements from the second collection into the first. If you specify that no duplicates are allowed, then the resulting collection will not have duplicates. If you specify true for "newCollection" then a new collection is returned containing the elements from both collections.

Parameters
c1:ICollectionView — the first collection
 
c2:ICollectionView — the second collection
 
allowDuplicates:Boolean (default = true) — if false then duplicate entries won't be added. Note that duplicates aren't checked for in the existing c1 collection.
 
newCollection:Boolean (default = false) — if true then a new ArrayCollection is created and returned, and the input collections aren't changed
 
allowNullValues:Boolean (default = false) — if true then null values are allowed in the output collection

Returns
ICollectionView — the same c1 collection that now contains c2's elements as well, OR a new ArrayCollection if newCollection is true
