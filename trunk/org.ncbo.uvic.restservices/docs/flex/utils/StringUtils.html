extractSentence()method 
public static function extractSentence(text:String, start:int = 0, end:int = 0, includePeriod:Boolean = true):String

Extracts a sentence from the given text. It scans backwards from the start index until it finds a period or the start of the string. Then it scans forwards from the end (if defined, otherwise from the start) until another period is found or the end of the string. The found sentence is returned (or an empty string). The ending period is included by default.

Parameters
text:String — the string to search
 
start:int (default = 0) — the starting index to search from
 
end:int (default = 0) — the optional ending index to search from
 
includePeriod:Boolean (default = true) — if true (default) then the returned sentence includes the ending period if present

Returns
String — the extracted sentence, or blank
