Classpublic class FilterBox
InheritanceFilterBox mx.containers.Canvas

An implementation of a filter text box with a clear button. It is very similar to the common filter textboxes seen in a lot of Eclipse views. When the filter text changes first a TextEvent is fired containing the changed filter text. Then after a short delay (which you can set) the ArrayCollection will be refreshed which causes the data to be filtered. Note that you can specify multiple fields to use for checking if an item is filtered. This might be useful in a DataGrid when you want to filter on all columns.

filterFunction()method 
public function filterFunction(item:Object):Boolean

This is the filter function used to filter the ArrayCollection. It simple calls the StringFilter.test() function with the string representation of the item. If the filterFields property is set, and the item has those properties then that value is used, otherwise item.toString() is used.
