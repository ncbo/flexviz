Packageflex.utils.ui
Classpublic class Spinner
InheritanceSpinner mx.core.UIComponent

Draws an animated spinner in the shape of a circle. A timer is used to animate the spinner. If the startImmediately property is set to true, then the timer is started when the spinner is added to its parent It is always stopped when it is removed from its parent. Otherwise you can call the stop() function or set running = false.
