Packageflex.utils.ui.renderers
Classpublic class ThumbnailCheckBox
InheritanceThumbnailCheckBox CheckBox mx.controls.CheckBox

Extends CheckBox to allow a renderer to draw a thumbnail between the checkbox and the text. The renderer (IThumbnailRenderer) specifies the width and height of the thumbnail that will be rendered. The renderer gets called after the checkbox has drawn itself.
