Packageflex.utils.ui
Classpublic class TextHighlighter

Highlights a portion of text in a IUITextField. By default the highlighting is done by bolding the text. But it can optionally make the matching text italic, underlined, different font size, different color. Either the specific text to highlight must be defined, or alternatively a function which returns the text to highlight must be defined.
