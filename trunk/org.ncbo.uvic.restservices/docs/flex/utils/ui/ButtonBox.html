Classpublic class ButtonBox
InheritanceButtonBox ResizableCanvas mx.containers.Canvas
SubclassesRadioButtonBox

Contains a vertical list of buttons (Button, CheckBox, RadioButton) which act similarily to a List, has the same selectedIndex, selectedItem, selectedIndices, selectedItems properties. The ButtonBox can have a custom itemRenderer as long as it is of type IDataRenderer. An itemsChanged event (flex.utils.ui.events.ItemsChangedEvent.ITEMS_CHANGED) is fired when one or more of the button's selected properties change, or when the user clicks on a button. This event is batched, so if all buttons are selected, then only one event is fire.

itemsChangedevent  
Event object type: flex.utils.ui.events.ItemsChangedEvent

This event is fired when a single button is selected by the user. It also gets fired when any of the selectedIndex, selectedItem, selectAll(), or selectNone() properties change. In this case only ONE event is fired after the change occurred and contains all the changed items.
