Classpublic class BorderLabel
InheritanceBorderLabel mx.controls.Label

Extends Label to provide styles for the border. Styles borderStyle="solid" borderColor="0x57BDFE" borderThickness="1" borderAlpha="1" cornerRadius="0" cornerRadii="null" If you change the borderThickness to more than 1 you will want to adding paddingLeft, paddingTop, paddingRight, and paddingBottom to the same value.
