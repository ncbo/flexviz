Classpublic class ResizableBox
InheritanceResizableBox mx.containers.Box
SubclassesOntologyList

Extends the Box class to let the user resize the box by dragging on a small 16x16 resize handle in the bottom right corner of the box. See the ResizeManager class for more details. You can also specify the minWidth, minHeight, maxWidth, and maxHeight properties to restrict the size of the box. 
    <ResizableBox
    Styles
    resizeHandleColor="0x666666"
    resizeHandleAlpha="1"
  >
        ...
        child tags
        ...
    </ui:ResizableBox>
