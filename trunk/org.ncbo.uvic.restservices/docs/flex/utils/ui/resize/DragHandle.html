Classpublic class DragHandle
InheritanceDragHandle mx.core.UIComponent

Renders a drag handle by drawing a set of small dots (squares). You can either specify an explicit width and height, or you can specify the number of rows and columns (the default is 1 column and 4 rows). You can also choose how big you want the dots to be. There are styles for setting the backgroundColor, backgroundAlpha (0 by default), dragHandleColor (0x666666 by default) and dragHandleAlpha (1 by default). There is also a dropShadowEnabled property that sets whether a DropShadowFilter is used (defaults to false).
