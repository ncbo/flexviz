Packageflex.utils.ui.resize
Classpublic class ResizableTitledBorderWindow
InheritanceResizableTitledBorderWindow TitledBorderWindow mx.containers.TitleWindow

Extends the TitledBorderWindow class to let the user resize the panel by dragging on a small 16x16 resize handle in the bottom right corner of the panel. See the ResizeManager class for more details. You can also specify the minWidth, minHeight, maxWidth, and maxHeight properties to restrict the size of the panel.
