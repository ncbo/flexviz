Packageflex.utils.ui.resize
Classpublic class MoveManager

Similar to the ResizeManager, this class adds support for moving a component by dragging it with the mouse. It also supports showing a custom cursor while dragging.

public function addMoveSupport(moveComponent:UIComponent, dragComponent:UIComponent = null, moveHandleParent:UIComponent = null):void

Adds support for moving a component.

Parameters
moveComponent:UIComponent — the component that will have its x and y values changed
 
dragComponent:UIComponent (default = null) — the component that will have a mouse_down listener added to listen for when the user drags it. If null then the moveComponent is used instead.
 
moveHandleParent:UIComponent (default = null) — the parent component which will have the move handle added to it.
