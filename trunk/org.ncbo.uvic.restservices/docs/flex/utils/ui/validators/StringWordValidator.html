Packageflex.utils.ui.validators
Classpublic class StringWordValidator
InheritanceStringWordValidator mx.validators.StringValidator

Extends the mx.validators.StringValidator class to also have support for restricting the minimum and maximum number of words allowed in a String. It also adds another property "trim" which will trim the input string before the validation occurs.

countWords()method
public static function countWords(txt:String):uint

Splits the text into words and returns the number of words found. If the text is null or blank (trimmed) then 0 is returned. The word separator regular expression is /\s+/g which splits the text into words based on whitespace.
