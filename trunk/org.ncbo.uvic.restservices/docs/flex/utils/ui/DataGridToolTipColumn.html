Classpublic class DataGridToolTipColumn
InheritanceDataGridToolTipColumn mx.controls.dataGridClasses.DataGridColumn

Extends DataGridColumn to support a headerToolTip property. It uses a custom headerRenderer class (which extends Label) and sets the tooltip on that renderer. You can also position the tooltip to: above, below, to the left or right of the header, or inline (on top of the header). Or if you don't specify a position it will use the default position which is slightly below and to the right of the mouse cursor. You can also set the headerTextTruncate flag which will truncate the headerText to fit the header, adding the ellipsis ("...") if necessary. Note that headerTextTruncate only works if headerTextWordWrap is false.
