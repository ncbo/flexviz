Classpublic class TitledBorderWindow
InheritanceTitledBorderWindow mx.containers.TitleWindow
SubclassesResizableTitledBorderWindow

This class extends TitleWindow to draw a solid border instead of the default Panel border. This border defaults to a solid square 1 pixel black border that goes around the entire panel except for where the title, status and close button are located. A border is also drawn between the title TextField and the status TextField. It also has support for added extra buttons in the top right corner of the titlebar, to the left of the close button.
