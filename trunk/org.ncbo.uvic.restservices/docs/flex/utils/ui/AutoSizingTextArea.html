Classpublic class AutoSizingTextArea
InheritanceAutoSizingTextArea mx.controls.TextArea

This TextArea automatically sizes itself. The vertical scroll policy is set to "off" by default. It resizes when the text changes, as well as when the TextArea styles change. Original idea came from this website: "Flex 3 Expanding/Collapsing TextArea" http://www.brentlamborn.com/post/Flex-3-Expanding-Collapsing-TextArea.aspx TextArea: http://livedocs.adobe.com/flex/3/langref/mx/controls/TextArea.html
