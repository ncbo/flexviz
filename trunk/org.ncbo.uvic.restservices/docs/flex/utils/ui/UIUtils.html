Useful method for creating a popup content window which displays a label and a checkbox.

Parameters
label:String (default = "") — the parent for the popup
 
title:String (default = "Confirm") — if the popup should be modal (defaults to true)
 
cbLabel:String (default = "") — the title for the window (defaults to "Confirm")
 
cbSelected:Boolean (default = false) — the content label (defaults to "")
 
flags:uint (default = 0x000C) — the label for the checkbox (defaults to "")
 
defaultBtnFlag:uint (default = 0x0004) — if the checkbox should be selected (defaults to false)
 
parent:DisplayObject (default = null) — the function which gets called when the window closes. It contains two boolean parameters: first if ok was pressed, and second if the checkbox was selected
 
callback:Function (default = null) — the width of the text - set this to force line wrapping
 
textWidth:Number (default = 0) — the width for the window, set to 0 if you want it to autosize itself
 
windowWidth:Number (default = 0)

Returns
