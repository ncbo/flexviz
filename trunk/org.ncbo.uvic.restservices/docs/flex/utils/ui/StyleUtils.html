Classpublic class StyleUtils

getNumberStyle()method 
public static function getNumberStyle(ui:UIComponent, styleName:String, defaultValue:Number = 1, min:Number, max:Number):Number

Tries to get a Number style value from the UIComponent. If the style doesn't exist then it returns the default value. It can also optionally restrict the number between a min and max. This function will only return NaN if your default value is NaN.
