Classpublic class URLLinkButton
InheritanceURLLinkButton mx.controls.LinkButton

Extends the LinkButton class to add some new functionality and make it more similar to the usual HTML links. You specify a URL and optionally a window name and when the link is clicked it will open that url in the given window using the navigateToURL function. By default the window name is null meaning it will open in a new window. You can also use the html window names like "_self" (same window), "_blank" (new window), "_parent" (parent window), or "_top" (top frame). See http://livedocs.adobe.com/flex/3/langref/flash/net/package.html#navigateToURL%28%29 for more details. There is also a right-click context menu that has two menu items: "Open Link in a New Window or Tab" and "Copy Link URL". This link defaults to being underlined and blue.

windowproperty 
public var window:String = "null"

Specifies the window name to use when the link is clicked. If null (default) then the link is opened in a new window. If "_self" then the link is opened in the same window. You can use custom window names too and the link will always open in that window.

This property can be used as the source for data binding.
