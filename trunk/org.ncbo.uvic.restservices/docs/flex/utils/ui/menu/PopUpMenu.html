Packageflex.utils.ui.menu
Classpublic class PopUpMenu
InheritancePopUpMenu mx.controls.Menu

Extends Menu to act like the Menu in the PopUpMenuButton class so that you can attach a menu to any UIComponent. You control how and when this menu pops up, either on a mouse click, double click, hover, etc. Most of the functions in this class were copied from the PopUpButton class and modified to suit this purpose. The source property is required, and should be set to the UIComponent that triggers the showing of this PopUpMenu. This menu will be positioned under the source component (unless it doesn't fit in which case it will be above).
