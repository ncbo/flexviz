Packageflex.utils.ui.menu
Classpublic class MenuItem
SubclassesMenu

Defines the basic properties of a menu item. MenuItem objects don't contain children. They can be one of the following types: - normal - check - radio - separator A MenuItem object also has an optional click handler. It also has a generic data object property.
