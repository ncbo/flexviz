Classpublic class TileListEx
InheritanceTileListEx mx.controls.TileList

Extends TileList to add the following styles: - horizontal and vertical gap between each item in the tile list - selection border color, thickness, and alpha - the border to draw when an item is selected - highlight border color, thickness, and alpha - the border to draw when an item is highlighted Styles horizontalGap="1" verticalGap="1" selectionBorderColor="0x009eff" selectionBorderThickness="2" selectionBorderAlpha="1" rollOverBorderColor="0x999999" rollOverBorderThickness="1" rollOverBorderAlpha="1"
