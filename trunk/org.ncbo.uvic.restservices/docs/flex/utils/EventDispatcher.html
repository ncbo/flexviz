Packageflex.utils
Classpublic class EventDispatcher
InheritanceEventDispatcher flash.events.EventDispatcher

Extends the default EventDispatcher class to add support for pausing the dispatcher to prevent events from being fired. It can also delay events and cause them to be fired later.

public function stopFiringEvents(delayEvents:Boolean = false):void

Stops events from being fired. If delayEvents is true then every event that would have been dispatched is saved and will be dispatched when startFiringEvents() is called.
