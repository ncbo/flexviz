public static function padZeros(num:Number, len:int = 2, left:Boolean = true, radix:int = 10):String

Pads a number with zeroes, either at the front (default) or the end.

Parameters
num:Number — the number to convert to a String and pad with zeroes
 
len:int (default = 2) — the required length of the resulting string, zeroes are padded to the number to make a string up to this length.
 
left:Boolean (default = true) — if the zeroes are added on the front of the number string.
 
radix:int (default = 10) — the radix used to convert the number to a string, defaults to decimal (10)

Returns
